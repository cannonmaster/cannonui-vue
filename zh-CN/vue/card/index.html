Card 商品卡片

<script lang="ts">
import { reactive } from 'vue';

export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },

  setup(props) {
    const { lang } = props;
    const state = reactive({
      imgUrl: 'https://placehold.co/300x300',
      title:

      price: '388',

      state

  }

</script>

shopName销售该产品的商店的名称String''

插槽

Card 组件提供了几个命名插槽，用于额外的自定义：

  • prolist: 产品列表。
  • price: 产品价格信息。
  • origin: 产品原始价格信息。
  • shop-custom-desc: 销售产品的商店的自定义描述。
  • footer: 卡片的页脚。
