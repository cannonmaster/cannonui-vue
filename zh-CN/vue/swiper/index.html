Swiper组件

On this page

  • 使用示例
  • Props
  • Events
  • Slots

这是一个用于创建 Swiper/轮播图的 Vue.js 组件。它提供了自定义滑块的选项，包括方向、循环、自动播放、分页等等。

<template>

        <cannonui-swiper
          direction="vertical"
          :init-page="page"
          :pagination-visible="true"
          pagination-color="cyan"

          style="height: 150px"
        >
          <cannonui-swiperitem
            v-for="item in list1"
            :key="item"
          >

              alt=""

<script lang="ts">
import { onMounted, reactive, ref, toRefs } from 'vue';
export default {
  props: {},
  emits: [],
  components: {},
  setup(props) {
    const swiper = ref(null);
    const state = reactive({
      page: 2,
      current: 1,
      list: [
        'https://picsum.photos/id/28/750/300',
        'https://picsum.photos/id/26/750/300',
        'https://picsum.photos/id/30/750/300',
        'https://picsum.photos/id/32/750/300'
      ],
      list1: [] as string[]
    });
    const change = (index: number) => {
      state.current = index + 1;
    };
    onMounted(() => {
      state.list1 = state.list.slice();
    });
    return {
      ...toRefs(state),
      swiper,
      change
    };
  }
};
</script>

Props

属性名类型默认值描述
widthNumber/Stringtypeof window === ‘object’ ? window.innerWidth : 375滑块容器的宽度。
heightNumber/String0滑块容器的高度。
directionString‘horizontal’滑块的方向。可以是’horizontal’（水平）或’vertical’（垂直）。
isStopPropagationBooleanfalse是否停止触摸事件传播。
paginationColorString‘#fff’分页点的颜色。
paginationVisibleBooleanfalse是否显示分页点。
loopBooleantrue是否循环播放。
durationNumber/String500幻灯片动画的持续时间（以毫秒为单位）。
autoPlayNumber/String0自动播放幻灯片之间的时间间隔（以毫秒为单位）。如果设置为 0，则禁用自动播放。
initPageNumber/String0当加载滑块时要显示的初始页面。
touchableBooleantrue滑块是否可触摸。
isPreventDefaultBooleantrue是否阻止默认触摸行为。
isCenterBooleanfalse是否将当前活动幻灯片居中显示。

Events

事件名描述
change当活动幻灯片更改时发出。将新活动索引作为参数传递给事件处理程序函数。

Slots

插槽名描述
默认幻灯片的内容。
page使用此插槽自定义分页点。如果使用此插槽，则默认分页点将被隐藏。
