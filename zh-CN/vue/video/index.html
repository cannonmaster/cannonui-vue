视频组件介绍

On this page

  • 示例用法

CannonUI Video 组件是一个 Vue.js 组件，可以在 Web 应用程序中渲染视频内容。

示例用法

Video

简单使用

自动播放

以下是使用视频组件的示例：

<style lang="scss" scoped></style>
<template>
  <div class="phone">
    <div class="phone-top">Video</div>
    <div class="phone-content">
      <div class="cannonui-video">
        <h2 class="cmp-usage__title">{{ h2_title.basic }}</h2>
        <cannonui-video
          :source="src"
          :type="type"
          :options="options1"
        ></cannonui-video>
        <h2 class="cmp-usage__title">{{ h2_title.auto }}</h2>
        <cannonui-video
          :source="src"
          :type="type"
          :options="options2"
        ></cannonui-video>
      </div>
    </div>
  </div>
</template>
<script lang="ts">
interface Translation {
  [key: string]: {
    h2_title: {
      basic: string;
      auto: string;
    };
  };
}
const translation: Translation = {
  en: {
    h2_title: {
      basic: 'basic',
      auto: 'auto play'
    }
  },
  'zh-CN': {
    h2_title: {
      basic: '简单使用',
      auto: '自动播放'
    }
  }
};
import { reactive, toRefs } from 'vue';
export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },
  emits: [],
  components: {},
  setup(props) {
    const { lang } = props;
    const state = reactive({
      src: '/src/samplev.mp4',
      type: 'video/mp4',
      options1: {
        autoplay: false,
        volumn: '0.5',
        poster: '',
        loop: false,
        controls: true,
        muted: false,
        disabled: false,
        playsinline: false,
        preload: '',
        touchPlay: false
      },
      options2: {
        autoplay: true,
        volumn: '0.5',
        poster: '',
        loop: false,
        controls: true,
        muted: true,
        disabled: false,
        playsinline: false,
        preload: '',
        touchPlay: 'true'
      }
    });
    return {
      ...translation[lang],
      ...toRefs(state)
    };
  }
};
</script>

Props

CannonUI Video 组件接受以下 props：

Prop 名称类型默认值必填项描述
sourceString''否要播放的视频文件的 URL。
typeString''否视频文件的 MIME 类型。
optionsObject{}是一个对象，指定视频播放器的各种选项，例如自动播放、音量和循环。

options 对象

作为 prop 传递的 options 对象应该具有以下结构：

属性类型默认值描述
autoplayBooleanfalse指定视频是否应该自动开始播放。
volumeNumber0.5指定视频的音量，介于 0 和 1 之间。
posterString''指定要在视频下载或播放之前显示的图像的 URL。
loopBooleanfalse指定视频是否应该在结束后从开头再次播放。
controlsBooleantrue指定视频播放器是否应该显示控件，如播放/暂停按钮。
mutedBooleanfalse指定视频是否应该被静音。
disabledBooleanfalse指定视频播放器是否应该被禁用。
playsinlineBooleanfalse指定视频是否应该在移动设备上内联播放。
touchPlayBooleanfalse指定当用户触摸屏幕时，视频是否应该在触摸设备上播放。
preloadString''指定视频是否应该预加载以及如何预加载。可能的值为“none”、“metadata”和“auto”。

事件

play视频开始播放时触发。
pause视频暂停时触发。
