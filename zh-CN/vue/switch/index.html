开关组件介绍

On this page

  • 效果展示

开关组件是一种可重用的 UI 元素，可用于表示 Web 应用程序中的开/关或真/假状态。提供的代码是开关组件的 Vue 3 实现，可以轻松集成到 Vue 应用程序中。

下面是使用卡片组件的示例：

<script lang="ts">
import { ref } from 'vue';
export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },
  emits: [],
  components: {},
  setup(props) {
    const isAsync = ref(false);
    const change = (value: Boolean, e: any) => {
      console.log('change function called');
    };
    const changeAsync = (value: Boolean, e: any) => {
      setTimeout(() => {
        isAsync.value = !isAsync.value;
      }, 2000);
    };
    const checked = ref(false);
    return {
      checked,
      isAsync,
      changeAsync,
      change
    };
  }
};
</script>

<template>
  <h2>Basic</h2>
  <cannonui-switch v-model="checked" />
  <h2>Disabled</h2>
  <cannonui-switch
    disable
    v-model="checked"
  />
  <h2>onChange</h2>
  <cannonui-switch
    v-model="checked"
    @change="change"
  />
  <h2>Async</h2>
  <cannonui-switch
    :model-Value="isAsync"
    @change="changeAsync"
  />
  <h2>Customize color</h2>
  <cannonui-switch
    v-model="checked"
    activeColor="cyan"
  />
  <h2>Customize text</h2>
  <cannonui-switch
    v-model="checked"
    activeText="开"
    inactiveText="关"
  />
</template>

属性

属性名称类型默认值描述
modelValueString 或 Booleanfalse开关的当前值
activeValueString，Number 或 Booleantrue开关处于激活状态时的值
inactiveValueString，Number 或 Booleanfalse开关处于非激活状态时的值
loadingBooleanfalse开关是否处于加载状态
disableBooleanfalse开关是否被禁用
activeColorString""开关处于激活状态时的背景色
inactiveColorString""开关处于非激活状态时的背景色
activeTextString""开关处于激活状态时显示的文本
inactiveTextString""开关处于非激活状态时显示的文本

事件

事件名称负载描述
update:modelValueString，Number或Boolean每当开关组件的值发生变化时，将发出此事件。
changeString，Number或Boolean和Event每当开关组件的值发生变化时，将发出此事件。
