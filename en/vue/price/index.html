Introduction

On this page

  • Props

The CannonUI Price component is a reusable UI component that displays a formatted price. It supports customization of symbol, decimal point, and strike-through styling.

Example Usages

Skeleton

Basic

10,000 . 00

Thousand

$10,000 . 00

No thousand

$10000 . 00

Here’s an example of how to use the Price component:

<style lang="scss" scoped></style>
<template>
  <div class="position-center">
    <div class="phone">
      <div class="phone-top">price</div>
      <div class="phone-content">
        <h2>{{ h2_title.basic }}</h2>
        <cannonui-price
          :thousand="true"
          :price="10000"
          :need-symbol="false"
        />
        <h2>{{ h2_title.thousand }}</h2>
        <cannonui-price
          :thousand="true"
          :price="10000"
          :need-symbol="true"
        />
        <h2>{{ h2_title.nothousand }}</h2>
        <cannonui-price
          :thousand="false"
          :price="10000"
          :need-symbol="true"
        />
      </div>
      <div class="phone-bottom"></div>
    </div>
  </div>
</template>
<script lang="ts">
interface Translation {
  [key: string]: {
    h2_title: {
      basic: string;
      thousand: string;
      nothousand: string;
    };
  };
}
const translation: Translation = {
  en: {
    h2_title: {
      basic: 'Basic',
      thousand: 'Thousand',
      nothousand: 'No thousand'
    }
  },
  'zh-CN': {
    h2_title: {
      basic: '基本使用',
      thousand: '千分位',
      nothousand: '无千分位'
    }
  }
};
import { reactive, toRefs } from 'vue';
export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },
  emits: [],
  components: {},
  setup(props) {
    const { lang } = props;
    const state = reactive({
      ...translation[lang]
    });
    return {
      ...toRefs(state)
    };
  }
};
</script>

Props

Prop NameTypeDefault ValueDescription
isStrikeBooleanfalseAdds a strike-through style to the price if true
thousandBooleantrueFormats the number with commas to represent thousands
needSymbolBooleantrueDisplays the specified symbol next to the price if true
symbolString’$’The symbol to display next to the price
priceString or Number0The price value to display in the component
decimalPointNumber2The number of decimal points to display for the price value
