The Pick component is a customizable picker used to select values from a list of options. It provides a user interface for selecting items or values, such as dates, times, or custom options.

<style lang="scss" scoped></style>
<template>
  <div class="phone">
    <div class="phone-top">Pick</div>
    <div class="phone-content">
      <h2></h2>
      <cannonui-pick
        :columns="columns"
        title="123"
        @confirm="confirm"
      ></cannonui-pick>
    </div>
    <div class="phone-bottom"></div>
  </div>

<script lang="ts">
import { computed } from 'vue';
import Pick from '@cannonui/vue';
export default {
  props: {},
  emits: [],
  components: { Pick },
  setup(props) {
    const columns = computed(() => [
      { text: 'nanJing', value: 'NanJing' },
      { text: 'wuXi', value: 'WuXi' },
      { text: 'zangZu', value: 'ZangZu' },
      { text: 'beiJing', value: 'BeiJing' },
      { text: 'lianYunGang', value: 'LianYunGang' },
      { text: 'zheJiang', value: 'ZheJiang' },
      { text: 'jiangSu', value: 'JiangSu' }
    ]);
    const confirm = ({
      selectedValue,
      selectedOptions
    }: {
      selectedValue: string[];
      selectedOptions: any;
    }) => {
      console.log('confirm');
    };
    return {
      columns,
      confirm
    };
  }
};
</script>

Props

NameTypeDefaultDescription
modelValueArray of (string | number)[]The selected values.
titleString''The title displayed at the top of the picker.
cancelTextString''The text for the cancel button in the toolbar.
okTextString''The text for the OK/confirm button in the toolbar.
columnsArray of (PickerOption | PickerOption[])[][]The options for each column in the picker. Each column can be represented by a single PickerOption or an array of PickerOption objects in case of multiple options within the column.
threeDimensionalBooleantrueWhether the picker has three-dimensional scrolling. If set to true, the columns will scroll independently. If set to false, the columns will scroll together.
swipeDurationNumber | String1000The duration of the swipe animation in milliseconds.
showToolbarBooleantrueWhether to show the toolbar at the top of the picker.
visibleOptionNumNumber | String7The number of visible options in each column.
optionHeightNumber | String36The height of each option in pixels.

Events

NameDescription
cancelTriggered when the cancel button is clicked.
changeTriggered when the selected values change. The updated selected values are emitted as an event.
confirmTriggered when the OK/confirm button is clicked.
update:modelValueTriggered when the selected values change. The updated selected values are emitted using v-model.

Slots

NameDescription
topCustom content to be placed at the top of the picker, above the columns.
defaultCustom content to be placed below the columns and toolbar in the picker.
