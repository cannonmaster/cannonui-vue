## Props

The following props can be passed to the CannonUI Searchbar Component:

| Prop            | Type    | Default | Description                                                                                          |
| --------------- | ------- | ------- | ---------------------------------------------------------------------------------------------------- |
| clearable       | Boolean | true    | Whether to display a clear button in the search bar to clear the input field.                        |
| inputType       | String  | 'text'  | The type of input element to render.                                                                 |
| shape           | String  | ''      | The shape of the search bar. Can be set to 'round'.                                                  |
| label           | String  | ''      | The label text to display above the search bar.                                                      |
| disabled        | Boolean | false   | Whether the search bar is disabled.                                                                  |
| autoFocus       | Boolean | false   | Whether the search bar should be focused automatically when the page loads.                          |
| readonly        | Boolean | false   | Whether the search bar is read-only.                                                                 |
| background      | String  | ''      | The background color of the search bar.                                                              |
| inputBackground | String  | ''      | The background color of the input field in the search bar.                                           |
| maxLength       | Number  | 9999    | The maximum number of characters that can be entered in the search bar.                              |
| focusStyle      | Object  | {}      | An object containing CSS styles to apply to the search bar when it is focused.                       |
| modelValue      | String  | ''      | The value of the search bar.                                                                         |
| placeholder     | String  | ''      | The placeholder text to display in the search bar when there is no value.                            |
| inputAlign      | String  | 'left'  | The text alignment of the input field in the search bar. Can be set to 'left', 'center', or 'right'. |

## Events

The following events are emitted by the CannonUI Searchbar Component:

| Event name | Arguments | Description                                                           |
| ---------- | --------- | --------------------------------------------------------------------- |
| search     | value     | Emitted when the user presses the enter key or clicks the search icon |
