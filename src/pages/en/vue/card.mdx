The `Card` component is a reusable Vue.js component that renders a product card with an image, title, price, and shop information.

## Example Usage

<div class="phone">
  <div class="phone-top">卡片组件</div>
  <div class="phone-content">
    <CardVue client:visible />
  </div>
  <div class="phone-bottom"></div>
</div>

Here's an example of how to use the `Card` component:

```vue
<script lang="ts">
import { reactive } from 'vue';

export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },
  emits: [],
  components: {},
  setup(props) {
    const { lang } = props;
    const state = reactive({
      imgUrl: 'https://placehold.co/300x300',
      title:
        'wild Alaskan salmon is packed with omega-3s, vitamin D, and other essential nutrients for a delicious and nutritious addition to any meal ',
      price: '388',
      vipPrice: '378',
      shopDesc: 'self-operated',
      delivery: 'free delivery',
      shopName: 'The Fishmen Catch',
      shopCustomDesc:
        'Fresh and sustainably caught seafood sh  op. Wide variety of high-quality seafood. Knowledgeable staff to assist with cooking tips and recipes.',
      shopCustomFooter: 'custom footer',
      priceSymbol: '$',
      prolist: ['jumbo', 'wild-caught']
    });
    return {
      state
    };
  }
};
</script>

<template>
  <div class="phone">
    <div class="phone-top">Card</div>
    <div class="phone-content">
      <section class="cmp-usage__section--showcase">
        <cannonui-card
          :img-url="state.imgUrl"
          :title="state.title"
          :price="state.price"
          :vipPrice="state.vipPrice"
          :shopDesc="state.shopDesc"
          :delivery="state.delivery"
          :shopName="state.shopName"
          :symbol="state.symbol"
        ></cannonui-card>
      </section>
    </div>
    <div class="phone-bottom"></div>
  </div>
</template>

<style scoped></style>
```

## Props

| Name          | Type    | Default | Description                                           |
| ------------- | ------- | ------- | ----------------------------------------------------- |
| `imgUrl`      | String  | `''`    | The URL of the product image.                         |
| `title`       | String  | `''`    | The title of the product.                             |
| `symbol`      | String  | `''`    | The currency symbol used for the product price.       |
| `price`       | String  | `''`    | The price of the product.                             |
| `vipPrice`    | String  | `''`    | The VIP price of the product.                         |
| `shopDesc`    | String  | `''`    | The description of the shop that sells the product.   |
| `delivery`    | String  | `''`    | The delivery information of the product.              |
| `shopName`    | String  | `''`    | The name of the shop that sells the product.          |
| `isNeedPrice` | Boolean | `true`  | Whether to show the price information of the product. |

## Slots

The `Card` component provides several named slots for additional customization:

- `prolist`: The product list.
- `price`: The price information of the product.
- `origin`: The original price information of the product.
- `shop-custom-desc`: The custom description of the shop that sells the product.
- `footer`: The footer of the card.
