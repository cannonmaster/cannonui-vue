## Props

| Name       | Type    | Default     | Description                                                                                                                                                        |
| ---------- | ------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `color`    | String  | `undefined` | The background color of the button. Can be a hex code, RGB value, or CSS color name.                                                                               |
| `shape`    | String  | `"round"`   | The shape of the button. Can be `"round"` or `"square"`.                                                                                                           |
| `plain`    | Boolean | `false`     | Whether the button should be plain or not. If `true`, the button will have no background color and the text color will be the color specified in the `color` prop. |
| `loading`  | Boolean | `false`     | Whether the button should be in a loading state. If `true`, a loading spinner will be displayed and the button will be disabled.                                   |
| `disabled` | Boolean | `false`     | Whether the button should be disabled or not. If `true`, the button will be disabled and clicking on it will have no effect.                                       |
| `type`     | String  | `"default"` | The type of button. Can be `"default"`, `"primary"`, `"info"`, `"success"`, `"warning"`, or `"danger"`.                                                            |
| `style`    | Object  | `{}`        | Custom styles to apply to the button.                                                                                                                              |
| `size`     | String  | `"normal"`  | The size of the button. Can be `"large"`, `"normal"`, `"small"`, or `"mini"`.                                                                                      |
| `block`    | Boolean | `false`     | Whether the button should be a block-level element or not. If `true`, the button will take up the full width of its container.                                     |

## Events

| Name    | Description                                                                                                 |
| ------- | ----------------------------------------------------------------------------------------------------------- |
| `click` | Triggered when the button is clicked. The click event object is passed as a parameter to the event handler. |
