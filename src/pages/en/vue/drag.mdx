### Props

| Prop      | Type   | Description                                                                          |
| --------- | ------ | ------------------------------------------------------------------------------------ |
| attract   | bool   | Specifies whether to attract the dragged element to the boundary or not.             |
| direction | string | Specifies the direction in which the element can be dragged. Default value is 'all'. |
| style     | string | Specifies the style of the element.                                                  |
| boundary  | object | An object that specifies the boundary of the dragged element.                        |
