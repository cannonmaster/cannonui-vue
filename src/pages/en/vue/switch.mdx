### Props

| Prop Name     | Type                       | Default Value | Description                                            |
| ------------- | -------------------------- | ------------- | ------------------------------------------------------ |
| modelValue    | String or Boolean          | false         | The current value of the switch                        |
| activeValue   | String, Number, or Boolean | true          | The value of the switch when it is active              |
| inactiveValue | String, Number, or Boolean | false         | The value of the switch when it is inactive            |
| loading       | Boolean                    | false         | Whether the switch is in a loading state               |
| disable       | Boolean                    | false         | Whether the switch is disabled                         |
| activeColor   | String                     | ""            | The background color of the switch when it is active   |
| inactiveColor | String                     | ""            | The background color of the switch when it is inactive |
| activeText    | String                     | ""            | The text to display when the switch is active          |
| inactiveText  | String                     | ""            | The text to display when the switch is inactive        |

### Events

## Events

| Event Name          | Payload                                      | Description                                                               |
| ------------------- | -------------------------------------------- | ------------------------------------------------------------------------- |
| `update:modelValue` | `String`, `Number`, or `Boolean`             | This event is emitted whenever the value of the switch component changes. |
| `change`            | `String`, `Number`, or `Boolean` and `Event` | This event is emitted whenever the value of the switch component changes. |
