The Tag component is a UI element used to display short labels or tags to describe or categorize content.

### Example Usages

<TagVue client:visible />

<script lang="ts">
interface Translation {
  [key: string]: {
    h2_title: {

  en: {

### Props

| Name      | Type    | Default     | Description                                                                       |
| --------- | ------- | ----------- | --------------------------------------------------------------------------------- |
| type      | String  | `'default'` | The type of the tag. Can be `'primary'`, `'success'`, `'danger'`, or `'warning'`. |
| color     | String  | `''`        | The background color of the tag.                                                  |
| textColor | String  | `'default'` | The text color of the tag.                                                        |
| plain     | Boolean | `false`     | Whether the tag has a plain background with a colored border.                     |
| round     | Boolean | `false`     | Whether the tag has a rounded border.                                             |
| mark      | Boolean | `false`     | Whether the tag is highlighted or marked.                                         |
| closeable | Boolean | `false`     | Whether the tag has a close button.                                               |
| style     | Object  | `{}`        | Custom styles to be applied to the tag.                                           |

### Events

| Name  | Description                                 |
| ----- | ------------------------------------------- |
| close | Triggered when the close button is clicked. |
| click | Triggered when the tag is clicked.          |
