The Row and Col components are used to create flexible and responsive layouts in Vue.js applications. The Row component creates a horizontal row that can contain one or more Col components, which are used to define the content within the row.

### Example Usage

<div class="phone">
  <div class="phone-top">Layout</div>
  <div class="phone-content">
    <LayoutVue />
  </div>
  <div class="phone-bottom"></div>

### Props

The `Row` component has the following props:

| Prop    | Type          | Default       | Description                                                                                                         |
| ------- | ------------- | ------------- | ------------------------------------------------------------------------------------------------------------------- |
| type    | String        | 'flex'        | The layout mode. Can be set to 'flex' or 'grid'.                                                                    |
| gutter  | Number/String | 0             | The gap between columns.                                                                                            |
| justify | String        | 'center'      | The horizontal alignment of columns. Can be set to 'start', 'end', 'center', 'space-around' or 'space-between'.     |
| align   | String        | 'flex-center' | The vertical alignment of columns. Can be set to 'top', 'middle', 'bottom', 'stretch' or 'baseline'.                |
| wrap    | String        | 'nowrap'      | Whether to wrap the columns when they exceed the container width. Can be set to 'wrap', 'wrap-reverse' or 'nowrap'. |

The `Col` component has the following props:

| Prop   | Type          | Default | Description                                                                     |
| ------ | ------------- | ------- | ------------------------------------------------------------------------------- |
| span   | Number/String | '24'    | The number of columns occupied by the column.                                   |
| offset | Number/String | '0'     | The number of columns to offset the column from the left side of the container. |
