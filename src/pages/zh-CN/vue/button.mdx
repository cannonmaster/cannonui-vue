### 示例用法

  lang="zh-CN"
  client:visible

## Props

| 名称       | 类型   | 默认值      | 描述                                                                                                        |
| ---------- | ------ | ----------- | ----------------------------------------------------------------------------------------------------------- |
| `color`    | 字符串 | `undefined` | 按钮的背景颜色。可以是十六进制码、RGB 值或 CSS 颜色名称。                                                   |
| `shape`    | 字符串 | `"round"`   | 按钮的形状。可以是 `"round"` 或 `"square"`。                                                                |
| `plain`    | 布尔值 | `false`     | 按钮是否应该是 plain 样式。如果为 `true`，按钮将没有背景颜色，并且文本颜色将是在 `color` 属性中指定的颜色。 |
| `loading`  | 布尔值 | `false`     | 按钮是否应该处于加载状态。如果为 `true`，将显示加载动画，并且按钮将被禁用。                                 |
| `disabled` | 布尔值 | `false`     | 按钮是否应该被禁用。如果为 `true`，按钮将被禁用，单击按钮将没有任何效果。                                   |
| `type`     | 字符串 | `"default"` | 按钮的类型。可以是 `"default"`、`"primary"`、`"info"`、`"success"`、`"warning"` 或 `"danger"`。             |
| `style`    | 对象   | `{}`        | 要应用到按钮的自定义样式。                                                                                  |
| `size`     | 字符串 | `"normal"`  | 按钮的大小。可以是 `"large"`、`"normal"`、`"small"` 或 `"mini"`。                                           |
| `block`    | 布尔值 | `false`     | 按钮是否应该是块级元素。如果为 `true`，按钮将占据其容器的整个宽度。                                         |

## 事件

| 名称    | 描述                                                         |
| ------- | ------------------------------------------------------------ |
| `click` | 当单击按钮时触发。单击事件对象将作为参数传递给事件处理程序。 |
