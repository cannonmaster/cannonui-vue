标签组件是一种 UI 元素，用于显示短标签或标签来描述或分类内容。

### 示例

<TagVue
  lang="zh-CN"
  client:visible
/>

以下是如何使用该组件的示例：

<script lang="ts">

  [key: string]: {
    h2_title: {

    };

### 属性

| 属性名    | 类型    | 默认值      | 描述                                                                   |
| --------- | ------- | ----------- | ---------------------------------------------------------------------- |
| type      | String  | `'default'` | 标签类型。可以是 `'primary'`，`'success'`，`'danger'` 或 `'warning'`。 |
| color     | String  | `''`        | 标签的背景颜色。                                                       |
| textColor | String  | `'default'` | 标签文本的颜色。                                                       |
| plain     | Boolean | `false`     | 标签是否有带颜色的边框。                                               |
| round     | Boolean | `false`     | 标签是否有圆角边框。                                                   |
| mark      | Boolean | `false`     | 标签是否被突出显示或标记。                                             |
| closeable | Boolean | `false`     | 标签是否带有关闭按钮。                                                 |
| style     | Object  | `{}`        | 要应用于标签的自定义样式。                                             |

### 事件

| 事件名 | 描述                           |
| ------ | ------------------------------ |
| close  | 当关闭按钮被点击时触发的事件。 |
| click  | 当标签被点击时触发的事件。     |
