Row 和 Col 组件在 Vue.js 应用程序中用于创建灵活和响应式的布局。Row 组件创建一个水平行，可以包含一个或多个 Col 组件，用于定义行中的内容。

### Props

`Row`组件具有以下属性：

| 属性名  | 类型          | 默认值        | 描述                                                                                  |
| ------- | ------------- | ------------- | ------------------------------------------------------------------------------------- |
| type    | String        | 'flex'        | 布局模式。可设置为'flex'或'grid'。                                                    |
| gutter  | Number/String | 0             | 列之间的间隙。                                                                        |
| justify | String        | 'center'      | 列的水平对齐方式。可设置为'start'，'end'，'center'，'space-around'或'space-between'。 |
| align   | String        | 'flex-center' | 列的垂直对齐方式。可设置为'top'，'middle'，'bottom'，'stretch'或'baseline'。          |
| wrap    | String        | 'nowrap'      | 当列超出容器宽度时是否换行。可设置为'wrap'，'wrap-reverse'或'nowrap'。                |

`Col`组件具有以下属性：

| 属性名 | 类型          | 默认值 | 描述                 |
| ------ | ------------- | ------ | -------------------- |
| span   | Number/String | '24'   | 列占据的栅格数。     |
| offset | Number/String | '0'    | 列左侧的偏移栅格数。 |
