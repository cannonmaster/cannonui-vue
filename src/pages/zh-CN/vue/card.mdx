Card 组件是一个可复用的 Vue.js 组件，用于渲染一个带有图片、标题、价格和商店信息的产品卡片。

  </div>
  <div class="phone-bottom"></div>
</div>

## 代码演示

```vue
<script lang="ts">
import { reactive } from 'vue';

export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },
  emits: [],
  components: {},
  setup(props) {
    const { lang } = props;
    const state = reactive({
      imgUrl: 'https://placehold.co/300x300',
      title:
        'wild Alaskan salmon is packed with omega-3s, vitamin D, and other essential nutrients for a delicious and nutritious addition to any meal ',
      price: '388',
      vipPrice: '378',
      shopDesc: 'self-operated',
      delivery: 'free delivery',
      shopName: 'The Fishmen Catch',
      shopCustomDesc:
        'Fresh and sustainably caught seafood sh  op. Wide variety of high-quality seafood. Knowledgeable staff to assist with cooking tips and recipes.',
      shopCustomFooter: 'custom footer',
      priceSymbol: '$',
      prolist: ['jumbo', 'wild-caught']
    });
    return {
      state
    };
  }
};
</script>

<template>
  <div class="phone">
    <div class="phone-top">Card</div>
    <div class="phone-content">
      <section class="cmp-usage__section--showcase">
        <cannonui-card
          :img-url="state.imgUrl"
          :title="state.title"
          :price="state.price"
          :vipPrice="state.vipPrice"
          :shopDesc="state.shopDesc"
          :delivery="state.delivery"
          :shopName="state.shopName"
          :symbol="state.symbol"
        ></cannonui-card>
      </section>
    </div>
    <div class="phone-bottom"></div>
  </div>
</template>

<style scoped></style>
```

### 属性

| 参数          | 说明                                 | 类型    | 默认值 |
| ------------- | ------------------------------------ | ------- | ------ |
| `imgUrl`      | 产品图片的 URL                       | String  | `''`   |
| `title`       | 产品的标题                           | String  | `''`   |
| `symbol`      | 产品价格所使用的货币符号             | String  | `''`   |
| `price`       | 产品的价格                           | String  | `''`   |
| `vipPrice`    | 产品的 VIP 价格                      | String  | `''`   |
| `shopDesc`    | 销售该产品的商店的描述               | String  | `''`   |
| `delivery`    | 产品的运送信息                       | String  | `''`   |
| `shopName`    | 销售该产品的商店的名称               | String  | `''`   |
| `isNeedPrice` | 是否需要展示产品的价格信息（布尔值） | Boolean | `true` |

## 插槽

`Card` 组件提供了几个命名插槽，用于额外的自定义：

- `prolist`: 产品列表。
- `price`: 产品价格信息。
- `origin`: 产品原始价格信息。
- `shop-custom-desc`: 销售产品的商店的自定义描述。
- `footer`: 卡片的页脚。
