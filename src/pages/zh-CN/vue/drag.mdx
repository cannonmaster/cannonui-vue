此组件允许您在指定的方向和边界内拖动元素。

### 效果展示

<DragVue client:visible />

以下是如何使用拖拽组件的代码示例：

```vue
<script lang="ts">
import { onMounted, reactive, ref } from 'vue';
export default {
  props: {
    lang: {
      type: String,
      default: 'en'
    }
  },
  emits: [],
  components: {},
  setup(props) {}
};
</script>

<template>
  <div>
    <cannonui-drag direction="all">
      <span style="padding: 10px; background: cyan">drag</span>
    </cannonui-drag>
  </div>
  <div>
    <cannonui-drag direction="x">
      <span style="padding: 10px 20px; background: pink">x</span>
    </cannonui-drag>
  </div>
  <div>
    <cannonui-drag direction="y">
      <span style="padding: 10px 20px; background: orange">y</span>
    </cannonui-drag>
  </div>
  <div>
    <cannonui-drag
      attract
      direction="all"
    >
      <span style="padding: 10px 20px; background: cyan">abc123</span>
    </cannonui-drag>
  </div>
</template>

<style scoped lang="scss"></style>
```

### Props

| Prop      | Type   | Description                             |
| --------- | ------ | --------------------------------------- |
| attract   | bool   | 指定是否将被拖动的元素吸附到边界。      |
| direction | string | 指定可以拖动元素的方向。默认值为“all”。 |
| style     | string | 指定元素的样式。                        |
| boundary  | object | 指定被拖动元素的边界的对象。            |
